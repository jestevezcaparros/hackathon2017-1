Valo Example

This is a sample mobile application which we can use to interface with a Valo environment. What this application will do is send two kinds of information to Valo:

  • The mobile device's current location, to be sent periodically
  • A user's happiness rating with the event along with the mobile device's current location, to be sent when user selects a "Happiness" rating button

Before you use this application, may we ask you to fill up the registration details in the Registration tab. You will be asked to enter some details about yourself, as well as details of a Valo instance you want to use. Don't forget to save the details, after which you can come back to the Home page using the tabs below.
